HTML y otras utilidades

Algunos programas para HTML y otras utilidades.
Título Lenguaje Descripción
Utilidad para HTML C Puede numerar 'headings' en HTML, crear una tabla de contenidos, extraer elementos, 'pretty-printing', etc.
poxhtml Perl Convierte archivos XHTML a PO, los traduce, y los convierte de nuevo a XHTML.
hcalproxy C Se ejecuta como un proxy y convierte el microformato 'hCalendar' a 'iCalendar'.
ical2html C Crea un calendario en una tabla HTML a partir de un archivo 'iCalendar'. También incluye 'icalfilter' e 'icalmerge', para extraer eventos y fusionar múltiples archivos 'iCalendar'.
shuffle C Mezcla las líneas de un archivo. Hace más o menos lo contrario que 'sort'.
progress C Copia la entrada a la salida mientras que muestra una barra de progreso.
plan2vcs Perl Convierte formato plano a 'vCalendar' (usando 'korganizer')
mnbftosch.pl Perl Permite definir un formato basado en XML usando una gramática de contexto libre. Coge la gramática y genera un esquema XML equivalente.
htmldiff C Compara dos archivos HTML y destaca las diferencias entre sí.
unsign.awk/sign.awk GNU AWK Puede eliminar la firma digital de un mensaje de email o añadir una
